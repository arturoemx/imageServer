imageServer  1.0
Este paquete implementa un servidor de imágenes. Docho paquete cuenta con un programa que captura imágenes de una cámara y las ofrece a multiples cliente a traves de una conexion vía sockets. Las imagenes se les aplica una transformación proyectiva de tal manera que una región de un plano en la escena se ajuste perfectamente a la imagen capturada. Tambien a cada imagen se le puede aplicar una mascara, esta máscara se puede utilizar para añadir información predeterminada a cada imagen capturada.
